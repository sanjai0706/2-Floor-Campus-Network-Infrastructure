# Enterprise Campus Network Infrastructure using Cisco Packet Tracer

## Project Overview

This project demonstrates the design and implementation of a two-floor enterprise campus network using Cisco Packet Tracer. The network supports over 220 devices with secure communication through VLAN segmentation, inter-VLAN routing, DHCP, and wireless connectivity.

---

## Objectives

- Design a scalable enterprise network
- Implement VLAN-based network segmentation
- Configure inter-VLAN routing
- Enable automatic IP address assignment using DHCP
- Configure trunk links between switches
- Provide wireless connectivity
- Test network connectivity
